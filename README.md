# [Advent of Code 2024](https://adventofcode.com/2024)

Prepare for IDE import:
```shell
./mill mill.bsp.BSP/install
```

Compile all:
```shell
./mill __.compile
```

Run all tests:
```shell
./mill __.test
```

Format all code:
```shell
./mill mill.scalalib.scalafmt.ScalafmtModule/
```

Execute day 1:
```shell
./mill day01.run
```
